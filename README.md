Getting Started
--------------------------------------------------------------------------------

Get [uv] and install toupie

```bash
uv pip install "git+https://github.com/boisgera/toupie"
```

then start the server with

```bash
uv run toupie
```

[uv]: https://docs.astral.sh/uv/
